DESCRIPCION DEL PROGRAMA:
Este programa es un sistema gestor del gimnasio "GYM-Fort", que permite tomar asistencias, registrar socios, añadirles 
hasta 3 actividades, estadisticas del local, elegir el tipo de membresía, ver los registros de cada cosa y, además, 
controlar automáticamente las cuotas de los socios con el uso de actualizacion de archivos de manera secuencial 
según la fecha actual registrada en el programa.
INSTRUCCIONES DE EJECUCIÓN:
1) Ingresar al archivo main.py y ejecutar
2) Una vez ejecutado, los archivos se actualizan automaticamente
3) Se puede navegar por el menu de cualquier manera debido a que cargamos datos ficticios en los archivos para comprobar
su funcionalidad.
4) Opciones:
    1_ Socios y Actividades
      < 
        1_ Registrar Socios
        2_ Inscripcion a actividad
        3_Volver
    2_ Asistencia y Estadísticas
      <
        1_ Registrar Asistencias
        2_ Ingresar a Estadísticas
        3_Volver
    3_ Ver archivos
      <
        1_  Archivo de socios
        2_  Archivo de Cuotas y actividades
        3_  Archivo de Asistencias
        4_  Volver
    4_Salir
5) Si se desea comprobar la correcta actualizacion de archivos solo hay que cambiar la fecha puesta en el
archivo "fecha.txt".
6) Partimos de una premisa de no modificar los archivos manualmente, y para ello no se debe tener abierto ningun archivo al ejecutar.
